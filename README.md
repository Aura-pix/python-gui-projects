# Python GUI Projects

A collection of PyQt6 desktop applications demonstrating GUI development and event-driven programming.

## Projects Included

### 1. Digital Clock

- Real-time clock with custom LED-style font
- Updates every second
- Green display on black background

### 2. Stopwatch

- Start/Stop/Reset functionality
- Millisecond precision

### 3. Weather App

- Live weather data via API
- Temperature and conditions display
- Celsius and Farenheit display choice

### 4. QR Code Generator

- Generate QR codes from text/URLs
- Save as image files

### 5. Installations

pip install PyQt6 qrcode pillow requests

### 6. Usage

python digital_clock.py
python stopwatch.py
python weather_app.py
python qrcode_generator.py
